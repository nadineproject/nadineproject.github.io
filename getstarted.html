GET STARTED

Quick Start Guide

Install the dockerized version of Nadine to quickly get a feel of how she will work in your enviroment.

1. Install Docker Engine following the instructions here.

2. Clone Nadine to your machine.

git clone https://github.com/nadineproject/nadine.git

3. Launch Nadine.

docker-compose up
