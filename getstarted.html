GET STARTED

Quick Start Guide

Install the Nadine in a virtual machine to quickly get a feel of how she will work in your enviroment.

1. Install Vagrant following the instructions here.

2. Clone Nadine to your machine.

git clone https://github.com/nadineproject/nadine.git cd nadine

3. Start Nadine. (this will take some time to build the VM)

vagrant up

4. Launch a browser and go to: http://127.0.0.1:8989/. Authenticate using admin/admin.
